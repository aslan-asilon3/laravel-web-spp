- Framework Laravel 8
- Template AdminLTE-3
- Yajra Laravel DataTable Serverside
- Sweetalert 2 
- Laravel Fortify (autentikasi)
- Spatie Laravel Role-Permission (role & permission)
- Laravel DOMPDF (pdf)
